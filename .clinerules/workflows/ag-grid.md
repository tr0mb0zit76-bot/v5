# AG Grid with Vue 3 Development

## When to Use This Skill

- Creating complex data tables with sorting, filtering, and pagination
- Building editable grids or spreadsheets
- Implementing large datasets with virtual scrolling
- Adding enterprise features like row grouping, pivoting, or charts
- Creating custom cell renderers or editors using Vue components

## Installation

For AG Grid with Vue 3, install the following packages:

```bash
npm install ag-grid-vue3 ag-grid-community
# For enterprise features:
npm install ag-grid-enterprise
